Sobre Mim

Olá! Me chamo Luiz Bueno, tenho 17 anos e atualmente trabalho com suporte técnico e implantação de sistemas. Sou apaixonado por tecnologia e estou dando meus primeiros passos no mundo da programação. Atualmente estudo HTML, CSS e JavaScript, e compartilho minha jornada de aprendizado para me tornar um desenvolvedor completo. Gosto de organizar meu tempo entre estudo, trabalho e projetos pessoais, sempre buscando aprender algo novo todos os dias. Criei este portfólio para mostrar um pouco do que venho aprendendo e também como forma de acompanhar minha evolução. Meu objetivo é trabalhar com desenvolvimento de sistemas e, no futuro, atuar em projetos que impactem a vida das pessoas por meio da tecnologia. Se quiser acompanhar minha trajetória ou trocar ideias, estou sempre disponível!

O que eu estou estudando:

Estudos FRONT END

Atualmente estou focado em aprender HTML, CSS e JavaScript, que são as bases da construção visual dos sites. Com isso, estou entendendo como criar páginas responsivas, interativas e bem estruturadas para o usuário final.

Estudos BACK END

Ainda irei começar meus estudos em back-end, mas pretendo aprender tecnologias como Node.js, MySQL e conceitos de APIs, autenticação, banco de dados e segurança. O objetivo é entender como funcionam os bastidores das aplicações e como os dados são processados.

Estudos FULL STACK

O FullStack une o conhecimento de front-end e back-end. Meu foco como fullstack será dominar tanto a parte visual quanto a lógica por trás dos sistemas, conectando tudo para criar aplicações completas. Com o tempo, quero trabalhar com integrações entre front e back, usando ferramentas como Express.js, MongoDB, Git e frameworks modernos.
